**Sentinel** is a next-generation, edge-deployed computer vision system designed to automate traffic violation detection. Powered by **YOLOv8** and **EasyOCR**, Sentinel instantly processes live surveillance feeds to detect triple-riding, illegal parking, and speeding, while generating cryptographically verified evidence cards.

---

## 🚀 Key Features

- **Edge AI Processing:** Runs advanced neural networks (YOLOv8) directly on camera nodes for zero-latency inference and tracking.
- **Instant OCR:** Extracts license plates instantly using EasyOCR with custom pre-processing filters for high accuracy.
- **Secure Evidence:** Generates immutable, cryptographically hashed (SHA-256) evidence cards for municipal non-repudiation.
- **Enterprise Dashboard:** An ultra-responsive, client-side rendered React dashboard utilizing hardware-accelerated Framer Motion animations to visualize live telemetrics.

## 🏗️ Technical Architecture

### Backend: Python & FastAPI
A high-performance asynchronous Python backend serving AI models with minimal latency overhead. 
- `backend_api.py`: The core FastAPI microservice handling multipart form image uploads and returning JSON telemetry and base64-encoded annotated images.
- **Computer Vision:** YOLOv8 for object detection and tracking.
- **OCR:** EasyOCR for text extraction from license plates.

### Frontend: React & Tailwind CSS
A modern, "Stripe-meets-Linear" aesthetic UI built with React and Vite.
- **Tailwind CSS:** For highly customizable, utility-first styling (featuring glassmorphism and neon accents).
- **Framer Motion:** For fluid, 60fps spring animations and mount transitions.
- **Recharts:** For interactive, responsive telemetry dashboards.

## 🛠️ Installation & Setup

### Prerequisites
- Python 3.9+
- Node.js 18+
- npm or yarn

### 1. Backend Setup
Navigate to the project root and install the Python dependencies:

```bash
# Create a virtual environment (optional but recommended)
python -m venv venv
source venv/Scripts/activate  # On Windows

# Install dependencies
pip install -r requirements.txt

# Run the FastAPI server
uvicorn backend_api:app --host 0.0.0.0 --port 8000
```
*The backend will be available at `http://localhost:8000`.*

### 2. Frontend Setup
Open a new terminal, navigate to the `frontend` directory, and install the Node dependencies:

```bash
cd frontend

# Install dependencies
npm install

# Start the Vite development server
npm run dev
```
*The frontend dashboard will be available at `http://localhost:5173`.*

## 💻 Usage

1. Open your browser and navigate to the frontend dashboard (`http://localhost:5173`).
2. Review the **Overview** landing page to understand the architecture.
3. Click **Start Evaluation** to enter the **Live Inference** mode.
4. Upload a test traffic image (e.g., `sample_traffic.jpg`).
5. The system will process the image through YOLOv8, extract license plates with EasyOCR, and return cryptographically verified evidence cards in real-time.
6. Check the **Saved Evaluations** tab to view your local evaluation history.

---

## 🔒 Security
All evidence generated by Sentinel is cryptographically hashed using SHA-256 upon capture. This guarantees the integrity of the data pipeline and prevents tampering of evidence before it reaches municipal databases.
